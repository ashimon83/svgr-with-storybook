# SVGR with Storybook
A Sample of generating SVG Icon React components and Storybook with one command.

## Requirement
- with named export
- automatic generate Storybook's story files

## Libraries
- [SVGR](https://react-svgr.com/)
- [Storybook](https://storybook.js.org/)

## Usage
```
yarn
yarn icon:generate
yarn storybook
```
